# Strava GPT project skeleton

This is the starting point for the Strava tool for a custom GPT project.

Included are:

* The `package.json` and `package-lock.json` files to ensure dependency version matching.
* The `schema.json` file which contains a JSON schema to be handed to ChatGPT at a later step.
* The `strav-auth.js` file contains a small script to be used by the students for generating access keys for the Strava api.
